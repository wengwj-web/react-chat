# react-chat

> 招聘软件,牛人跟boss的聊天app

## Build Setup

``` bash
# install dependencies
npm install 

# serve with hot reload at localhost:3000/login
npm start 

# running the server
npm run server_bak

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
